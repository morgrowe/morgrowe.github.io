C.B. Stonemasonry

June 2013 | Website design

C.B. Stonemasonry is a growing stonemasonry business that takes on all aspects of stonemasonry. They're based in South Wales but work all over the United Kingdom. An online presence needed to be established to allow potential customers to contact the company, view their work and submit testimonials.

All the technical details including hosting, domains, email and maintenance are taken care of. This allows the business owner to request information to be changed over the phone.

The website was created using HTML, CSS and jQuery to provide users with a unique and memorable experience.

Feel free to visit C.B. Stonemasonry's website.

Technologies Used

  • HTML 5
  • CSS 3
  • JavaScript (jQuery)
  • Wufoo (forms)
