Ehcho.com Version 1

May 2012 | Website design

Ehcho.com was launched in May 2012. The original plan for the website was to write a few blog posts every couple of days and publish them over time. The website was designed, coded, integrated with WordPress in a couple of days. Version one looked quite bland because a new version was going to be released shortly after.

After discovering more WordPress features and techniques, new versions of the website were created behind the scenes. Eight designs were created that lead to version two.

A combination of work and real life commitments lead to the website becoming neglected. The website features a custom theme and some pages.

Technologies Used
